
这是一个用于在Android设备上播放PC音频的项目，主要原因是我的电脑上没有扬声器，我也不想买。
并且我的平板电脑有一个不错的扬声器。

请注意，这个项目的大部分代码是由AI生成的，不过我目前正在使用，所以基本功能是没有问题的。


如果你想编译这个项目，你需要
- 安装Visual Studio 2022，并且安装.NET桌面开发工作负载
- 安装Android Studio，并且安装Android SDK(36)


如果想使用，你需要使用USB连接你的Android设备，并且在设备上启用开发者选项和USB调试。
两端的程序会在启动后自动连接并开始传输音频。
